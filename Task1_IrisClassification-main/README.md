📌 Task 1 – Iris Classification (CodeAlpha Internship)

This project focuses on building a Machine Learning model to classify Iris flowers into three species: Setosa, Versicolor, and Virginica.

🚀 Project Overview

This project performs the following tasks:

Data loading and preprocessing

Exploratory Data Analysis (EDA)

Feature visualization

Train-test split

Machine Learning model training (Random Forest Classifier)

Model evaluation

Saving the model (iris_model.pkl)

Visualizations (confusion matrix, pair plot, feature importance)

🧠 Tech Stack Used

Python

NumPy

Pandas

Scikit-Learn

Matplotlib & Seaborn

Jupyter Notebook
